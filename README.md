   

## 📦 Inventory Management System  


### 🚀 Overview  
Inventory Management System ek powerful tool hai jo businesses ko unke stock, orders, aur suppliers ka sahi tarike se management karne me madad karta hai. Ye system user-friendly interface aur strong backend functionalities ke sath aata hai.

### 🛠️ Features  
✅ User Authentication (Login/Signup)  
✅ Product Management (Add, Update, Delete)  
✅ Stock Tracking & Notifications  
✅ Order Processing System  
✅ Supplier Management  
✅ Dashboard with Analytics  

### 🔧 Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** ASP.NET Core  
- **Database:** SQL Server  
- **Authentication:** Identity Framework  

### 🏗️ Installation Guide  

1️⃣ **Clone the repository**  
```bash
git clone https://github.com/Bhaneshvar007/Inventory-Management.git
cd InventoryManagement_System/InventoryManagement_System
```  

2️⃣ **Setup Database**  
- SQL Server install karein aur ek new database banayein.  
- Connection string ko `appsettings.json` me update karein.  

3️⃣ **Run the Application**  
```bash
dotnet restore
dotnet run
```  

4️⃣ **Access the Web App**  
Browser me open karein:  
```
http://localhost:5000
```
